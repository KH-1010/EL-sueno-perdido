      Console Guides(Step-by-Step Knowledge Base)

      Basic

      Transferring via PC / USB

      1. USB Preparation

      • • Format: FAT32 with MBR partition map.
      • • Packages: Place .pkg files on the root of the USB.
      • • Licenses: Create an exdata folder on the root for .rap files.

      2. Console Setup

      • • Port: Use the rightmost USB port (closest to the disc drive).
      • • Install: Go to Package Manager > Install Package Files > Standard.
      • • Note: HEN or CFW must be enabled before installing.
      Essential

      Activating Games (.RAP Files)

      Most PSN games require a license file (.rap) to boot. Without it, you will encounter a "Copyright Protection" or "Renew License" error.

      Installation Flow

      • • Place the .rap file inside the exdata folder on your USB.
      • • Keep the USB plugged in while launching the game for the first time.
      • • HEN Users: Ensure your console is activated (PSN Account linked).

      Pro Tips

      • • Filename: Do not rename .rap files; they must match the Content ID.
      • • Offline: Use Apollo Save Tool to activate licenses without PSN.
      Advanced

      Remote Installation (FTP/Network)

      Best for files over 4GB (FAT32 limit). Enable FTP in webMAN MOD or multiMAN, use an FTP client like FileZilla on your PC, and transfer PKGs to:
      dev_hdd0/packages. Once finished, install via Package Manager > Install Package Files > System Storage.

      Maintenance

      Fixing Missing Icons & Errors

      If game icons disappear or the XMB feels sluggish, you need to rebuild the database.

      • • Turn off the PS3™ and enter Safe Mode (Hold Power until you hear 2 quick beeps).
      • • Connect a controller via USB and select "Rebuild Database".
      • • Safe: This does not delete your games; it only refreshes the system index.
