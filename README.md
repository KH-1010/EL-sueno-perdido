# 🎮 مكتبة الألعاب العربية — Arabic PlayStation Games Library & PSN Database

موقع لمكتبة ألعاب PlayStation الكلاسيكية مع دعم الألعاب المدبلجة والمترجمة عربياً، مبني بهندسة الأداء الفائق وبدون أي تجمد (Zero Lag Architecture)، ويعمل مباشرة على **GitHub Pages** بدون الحاجة لأي خوادم أو Cloudflare!

---

## ✨ المميزات والترقيات الفاخرة (Latest Upgrades)

- 🌟 **مركز التحميل الذكي الشامل (Universal Smart Download Center):**
  - متكامل مع نافذة التحميل الفاخرة المستوحاة من متجر **PS5 / PSN Store**.
  - زر التحميل المباشر السريع من سيرفرات سوني الرسمية (Direct Sony Server CDN).
  - زر نسخ رابط التحميل المباشر (أو جميع روابط الأجزاء دفعة واحدة) للصقه في برامج التحميل السريع مثل **IDM** و **JDownloader**.
  - زر تحميل ملف الترخيص المطلوب (RAP License) ونسخ رابطه مباشرة للتشغيل على المحاكيات والأجهزة المعدلة.
- 🏷️ **أزرار التصنيفات السريعة (Quick Genre Pills Filter):**
  - تصفية فورية للألعاب بضغطة زر واحدة دون إعادة تحميل الصفحة (🌟 الكل | ⚔️ أكشن ومغامرات | 🧟 رعب وبقاء | ⚽ رياضة ومسابقات | 🏎️ سباق وتفحيط | 🥊 قتال ومصارعة | 🔮 تعاقب أدوار RPG).
- 📺 **وضع التلفاز القديم (Retro CRT Scanline Overlay):**
  - تجربة نوستالجيا حقيقية بضغطة زر لتفعيل خطوط المسح الضوئي (Scanlines & Chromatic Aberration) التي تحاكي شاشات التلفاز الكلاسيكية في التسعينيات وبداية الألفية.
- 🔊 **نغمات الإقلاع الكلاسيكية (PS1 & PS2 Boot Sound Chimes):**
  - أزرار تفاعلية لتشغيل أصوات بدء تشغيل بلايستيشن 1 وبلايستيشن 2 الأسطورية المولدّة تقنياً عبر **Web Audio API** بدون أي ملفات MP3 خارجية بحجم 0 بايت!
- 🚀 **مستشار التشغيل والمحاكيات (Emulation Advisor):**
  - بطاقة ذكية تفاعلية ترشد اللاعبين للمحاكي والإعدادات المثلى حسب جهازهم:
    - 🖥️ **كمبيوتر قوي (PC High-End):** دقة 4K / 60FPS عبر محرك Vulkan و Anisotropic Filtering 16x.
    - 💻 **كمبيوتر اقتصادي أو لابتوب (PC Low-End):** دقة 1x/2x واستخدام DirectX 11 لتسريع الأداء بدون تقطيع.
    - 📱 **هواتف أندرويد وتابلت:** توصيات محاكيات AetherSX2 و DuckStation.
    - 🕹️ **أجهزة محمولة (Steam Deck / ROG Ally):** ضبط استهلاك الطاقة (8W-10W TDP) لأداء سلس 60Hz.
    - 📺 **أجهزة كونسول أصلية (PS3 / PS2 Modded):** نقل الملفات عبر فلاش USB بصيغة FAT32 والتشغيل عبر Multiman أو OPL.
- 🔍 **محرك البحث الصوتي الذكي (Arabic Phonetic Fuzzy Search):**
  - يدعم البحث بالأسماء العربية الصوتية للعناوين الإنجليزية (مثل: *رزيدنت إيفل، كراش، جود أوف وور، تيكن، سايلنت هيل، فيفا، بيس...*).
- 🚀 **100% Static & GitHub Pages Ready:** لا يحتاج إلى Cloudflare Workers أو خوادم خلفية؛ يعمل مباشرة من المتصفح بسرعة فائقة.
- ⚡ **Zero Lag Engine:** معمارية التمرير الافتراضي (Virtual Scrolling & Pagination) المستوحاة من مشروع [PSN-Content](https://github.com/PS3-Pro/PSN-Content)، مع تحميل تدريجي بدون تجمد المتصفح حتى مع وجود أكثر من 60,000 لعبة!
- 🎙 **DUB:** قسم مخصص للألعاب المدبلجة عربياً مع شارات ذهبية فاخرة.
- 📝 **SUB:** قسم مخصص للألعاب المترجمة عربياً مع شارات خضراء نيونية.
- 🖼 **12,000+ غلاف:** ربط مباشر وسريع مع مكتبة Libretro الكلاسيكية.

## 🔄 الأتمتة عبر GitHub Actions

| Workflow | الوظيفة | الجدولة |
|----------|---------|---------|
| `pages.yml` | تحديث NPS + سكربت الأرشيف + نشر الموقع | يومياً الساعة 03:00 UTC |
| `deploy-site.yml` | نشر الموقع عند كل دفنة (push) | عند كل push لـ master/main |

### كيفية التشغيل يدوياً:
```bash
# تحديث قاعدة بيانات NPS
python update-nps.py

# تشغيل سكربت الأرشيف
node scraper.js
```

## 🌐 تفعيل ونشر الموقع عبر GitHub Pages

لتشغيل الموقع مباشرة على حسابك:
1. اذهب إلى إعدادات المستودع (Settings) ⟵ **Pages**.
2. تحت قسم **Build and deployment**، اختر **GitHub Actions** كمصدر (Source).
3. سيقوم الـ Workflow التلقائي ببناء ونشر الموقع فوراً على الرابط:
   **https://kh-1010.github.io/EL-sueno-perdido/**
